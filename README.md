# ggMcCabeSim

**ggMcCabeSim** is a standalone McCabe-Thiele Simulator for distillation column design for binary systems.<br> 
It is distributed as a Windows executable, so users can download it and run it immediately.<br>
This repository provides the compiled program (<strong>ggMcCabeSim.exe</strong>) for Win-10 64 bit or later.<br> 
A local help file (<strong>ggMcCabeSim_help.html</strong>) is also given for users who want to see what the program does.

Simulation, McCabe-Thiele, distillation column, binary distillation, column design

## Contents
- `ggMcCabeSim.zip` — contains the main executable for McCabe-Thiele Simulator  
- `ggMcCabeSim_help.html` — Program description, usage instructions and documentation

## Usage
Download **ggMcCabeSim.zip** from this repository or from the Releases page.<br>
Unzip the file to a directory of your local disk. It will create the executable **ggMcCabeSim.exe**.<br> 
Then run it.The first time it is Run, it will create the help file in current directory.<br>
 
Refer to `ggMcCabeSim_help.html` for detailed explanations of features, input formats, and examples.
